# Post-Install Guide

Welcome to your OpenCode AI assistant!

## Onboarding

Not sure where to start? Run the guided tour anytime:

```bash
# Interactive menu - choose your pace
~/.config/opencode/scripts/onboarding.sh

# Quick commands only
~/.config/opencode/scripts/onboarding.sh --quick

# Resume where you left off
~/.config/opencode/scripts/onboarding.sh --resume
```

The onboarding covers:
1. Welcome & overview
2. Your first memory
3. Setting reminders
4. System health checks
5. What to explore next

**You can skip anything and come back later.**

## First Steps

### 1. Test Memory
```
mem store "My first memory - installation complete"
mem search "installation"
```

### 2. Set a Reminder
```
remind me to check OpenCode in 5 minutes
```

### 3. Check System Health
```bash
~/.config/opencode/scripts/health-check.sh
```

### 4. Create a Blog Post (if blog installed)
```
blog create "My First Post" --content "Hello World"
```

