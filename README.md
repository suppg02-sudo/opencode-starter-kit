# OpenCode Starter Kit

A menu-driven setup system for OpenCode AI agent environments.

## What It Installs

### Core Systems
- **Skill Factory** - Meta-skill for creating/managing skills
- **Memory System** - PostgreSQL with pgvector for persistent AI memory
- **Tracking System** - Flow and context tracking

### Skills
- Cron management
- Reminder system
- Blog post creator (Hugo)
- Memory gateway
- Tracking gateway
- OpenRAG document retrieval

### Container Services
- Docker Compose base
- Portainer (container management)
- Homepage (dashboard)
- NextExplorer (file browser)
- UserMemos (notes)

## Usage

1. Clone this repo
2. Open in OpenCode
3. Type `setup`
4. Follow the interactive menu

## Requirements

- Linux server
- Docker installed
- Python 3.10+
- OpenCode agent
